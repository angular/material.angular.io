slider

Module: MdSliderModule

Directives

MdSlider

Allows users to select from a range of values by moving the slider thumb. It is similar in behavior to the native `<input type="range">` element.

Properties

Name Description
@Input()

disabled

Whether or not the slider is disabled.
@Input('thumbLabel')

thumbLabel

Whether or not to show the thumb label.

onTouched

onTouch function registered via registerOnTouch (ControlValueAccessor).
@Input()

step

The values at which the thumb will snap.
@Input()

tickInterval

How often to show ticks. Relative to the step so that a tick always appears on a step. Ex: Tick interval of 4 with a step of 3 will draw a tick every 4 steps (every 12 values).

tickIntervalPercent

The size of a tick interval as a percentage of the size of the track.

percent

The percentage of the slider that coincides with the value.
@Input()

value

Value of the slider.
@Input()

min

The miniumum value that the slider can have.
@Input()

max

The maximum value that the slider can have.
@Input()

invert

Whether the slider is inverted.
@Input()

vertical

Whether the slider is vertical.

invertAxis

Whether the axis of the slider is inverted. (i.e. whether moving the thumb in the positive x or y direction decreases the slider's value).

invertMouseCoords

Whether mouse events should be converted to a slider position by calculating their distance from the right or bottom edge of the slider as opposed to the top or left.

trackFillStyles

CSS styles for the track fill element.

ticksContainerStyles

CSS styles for the ticks container element.

ticksStyles

CSS styles for the ticks element.

thumbContainerStyles

direction

The language direction for this slider element.
@Output()

change

Event emitted when the slider value has changed.

Additional classes

MdSliderChange

A simple change event emitted by the MdSlider component.
